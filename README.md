# Enviro Web (Touch Grass)

Stand proud, you can code. Welcome to the official frontend repo for **Enviro**. 

While the backend team is in the trenches doing data science and soldering ESP32 sensors, this repo is where the aura is actually generated. Urban agriculture was out here playing terrorball for years, so we had to step in and bring the tiki-taka to the browser. We migrated this whole UI from vanilla HTML to a modern React architecture because we're putting up generational numbers and refuse to drop a disasterclass in production. 

Basically, we're connecting dirt to the cloud so your balcony plants can have a Prime Messi type of season. No cap.

## Tech Stack (ELITE BALL KNOWLEDGE)
We dropped the legacy tech and signed some absolute Galacticos for this build. Elite ball knowledge only:

* **React 18:** The playmaker. Generational talent. Dictating the tempo of the entire DOM.
* **Vite:** Pure pace merchant. The Mbappe of build tools. Bundles the code so fast it'll literally break your ankles. 
* **React Router DOM v6:** KDB-level vision. Threading the needle between pages without ever reloading the browser. 
* **Custom CSS:** We are raw-dogging Flexbox and CSS Grid. Zero Tailwind CSS AURA BRO!
* **Icons (Flaticon/Lucide):** Top bins aesthetics.
